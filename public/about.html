About

Information gathering is important, for blue and red teamers.

The information gathering process is repetitive and time consuming.

To work efficiently, some automation is required.

AutoRecon is an information gathering web application, that automates the process of gathering information.

AutoRecon finds subdomains, IP addresses, and open ports. The application uses the same sources as some of the popular tools in linux, including existing APIs, databases, and programming modules.

If you create an account, you will be able to save your search results for later use. When logged in, you will find them under History.
